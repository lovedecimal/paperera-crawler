# PaperEra 论文爬虫与数据分析

## 🌟 项目简介
本项目基于Python实现「计算机大模型」领域论文数据的可视化分析工具，核心功能是自动爬取（或模拟）PaperEra平台的论文信息，并生成高被引论文的被引量TOP10图表。项目兼顾**易用性**与**专业性**：小白可一键运行生成结果，同时代码包含工程化设计（如多系统兼容、数据容错），适配学术研究与技能展示场景。
