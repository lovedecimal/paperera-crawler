# PaperEra 论文爬虫与数据分析

## 🌟 项目简介
本项目基于 Python 实现「计算机大模型」领域论文数据的可视化分析工具。核心功能是自动爬取（或模拟）PaperEra 平台的论文信息，并生成高被引论文的被引量 TOP10 图表。项目兼顾 **易用性** 与 **专业性**：小白可一键运行生成结果，同时代码包含工程化设计（如多系统兼容、数据容错），适配学术研究与技能展示场景。

## 🚀 快速使用指南
### 1. 环境准备
-   **步骤 1**：安装 Python（推荐 3.14 版本），下载地址：[Python 官方下载页](https://www.python.org/downloads/)
    -   安装时 **必须勾选**「Add Python to PATH」（否则后续命令无法执行）；
    -   验证安装：打开命令提示符/终端，输入 `python --version`，显示版本号即安装成功；
-   **步骤 2**：打开「命令提示符（Windows）/终端（Mac/Linux）」，输入以下命令安装依赖库：
    ```bash
    pip install pandas matplotlib numpy
    ```

### 2. 运行代码生成图表
1.  **步骤 1**：将项目中的 `generate_fig.py` 文件保存到电脑任意文件夹（如桌面）；
2.  **步骤 2**：在文件所在文件夹空白处，按住「Shift 键 + 鼠标右键」，选择「在此处打开命令窗口 / Windows 终端」；
3.  **步骤 3**：在弹出的黑窗口中输入以下命令，按回车：
    ```bash
    python generate_fig.py
    ```
4.  **步骤 4**：等待 10 秒左右，窗口提示「✅ 真实数据版图表已保存至：XXX」即完成；
5.  **步骤 5**：打开文件夹中的 `figs` 目录，即可看到生成的 `论文被引量TOP10.png` 高清图表。
